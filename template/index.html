Servicios Instalados

Servicio ApacheServicio Web

Este es un servicio para brindar páginas web estáticas en su mayoría, es un utiliazo mundialmente debido a que es muy sencillo de instalar

Servicio DNSServicio Web

El servicio DNS proporciona una forma más sencilla de entrar a los sitios web, los cuales están alojados en un servidor, este DNS brinda un nombre sencillo para encontrar el servidor

Servicio RDPServicio Conexión Remota

Este servicio proporciona una forma de conectarse al servidor de forma remota con la particularidad de que es con interfaz gráfica, esto es muy útil para servidores Windows

Servicio SSHServicio Conexión Remota

Este servicio proporciona una manera de conectarse al servidor de forma remota pero solo por medio de Consola de comandos. Pueden utilizarse programas como putty para realizar esta conexión
